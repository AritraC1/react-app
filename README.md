## Getting Started

### Prerequisites
- Node.js (v16+ recommended)
- npm or yarn

### Installation
1. Clone the repository:
```bash
git clone https://github.com/AritraC1/react-app.git
cd react-app
```

2. Install dependencies and run:
```bash
npm install
npm run dev
```
